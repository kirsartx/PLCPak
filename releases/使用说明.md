# PLCPak 使用说明

**当前版本：** v1.0.74  
**适用对象：** 游戏资源压缩、去广告、回填网盘链接、生成发布文案、TG 发送等日常发布流程。

---

## 1. 安装与启动

1. 解压 `PLCPak_v1.0.74_WinUI.zip` 到任意目录（路径尽量不含特殊字符）。
2. **保持解压后的文件夹完整**，不要只复制 `exe` 出去。
3. 双击 **`PLCPak.WinUI\启动 PLCPak.bat`** 启动（推荐）。
4. 程序为自包含发布，一般**无需单独安装 .NET**。

**若无法启动：**

- 查看 `PLCPak.WinUI\logs\plcpak-startup.log`（或上级 `logs\plcpak-startup.log`）
- 确认 Windows 10/11 x64，且未被安全软件拦截

---

## 2. 目录结构

解压后根目录大致如下：

```
PLCPak_v1.0.74_WinUI/
├── PLCPak.WinUI/          GUI 主程序
│   ├── 启动 PLCPak.bat    ← 推荐启动方式
│   ├── 启动说明.txt
│   └── app/               程序本体（exe 与运行库）
├── PLCPak.Cli/            命令行版（可选）
├── data/                  配置、7-Zip、广告/密码样本
├── scripts/               维护脚本（如更新广告清单）
├── workspace/             任务工作区（你的数据在这里）
├── logs/                  运行日志
├── CHANGELOG.md           版本更新记录
└── 使用说明.md            本文件
```

**重要：** `workspace/` 存放任务、导入文件、导出结果等，升级版本时请一并备份或迁移。

## 3. 简易模式（默认，推荐新手）

首次启动会有动画引导；也可在 **设置 → 重新播放新手引导** 再看一遍。

### 三步上手

| 步骤 | 做什么 | 在哪里 |
|------|--------|--------|
| 1 | 拖入游戏文件夹/压缩包，快速压缩去广告 | 顶部 **快速处理**（Ctrl+1） |
| 2 | 新建任务或从帖子创建，按内嵌向导逐步完成 | **任务工作台**（Ctrl+2） |
| 3 | 查看待发布、TG 队列、健康问题 | **发布看板**（Ctrl+3） |

### 跟着提示走

- 窗口**底部「全局下一步」**会告诉你当前最该做什么，可一键跳转或执行。
- 选中任务后，右侧 **分步发布向导** 会引导：素材准备 → 链接回填 → 文案生成 → 发布。
- 需要更多功能时，打开右上角 **「专业」** 开关（`Ctrl+Shift+P` 可切换）。

### 简易模式导航

简易模式下顶部常驻：**快速处理 · 任务工作台 · 发布看板 · 设置**  
发布向导已并入任务页，按 `Ctrl+4` 也会进入任务工作台并聚焦向导区。

## 4. 推荐发布流程

```
从帖子创建任务（或新建任务）
    ↓
导入素材 / 拖入 inbox
    ↓
运行流水线（压缩、去广告、解压等）
    ↓
回填百度/夸克/TG 链接（Ctrl+S 保存）
    ↓
生成发布文案
    ↓
标记渠道已发布 / 发送 TG
    ↓
在看板确认队列与健康状态
```

**任务页向导快捷键：**

- `Ctrl+Alt+←` / `Ctrl+Alt+→`：上一步 / 下一步
- `Ctrl+Alt+Enter`：执行当前向导步骤
- `Ctrl+Enter`：运行完整流水线
- `Ctrl+S`：保存发布链接

## 5. 各页面说明

### 快速处理（Ctrl+1）

- 将源文件夹或压缩包**拖入**列表即可加入队列。
- 适合「只想压一包、不去建任务」的场景。
- 压缩配置在 **设置** 中调整。

### 任务工作台（Ctrl+2）

- 左侧：任务列表、搜索筛选、新建/导入。
- 右侧：任务详情 + **内嵌分步发布向导**。
- 可多选任务做批量归档、恢复、删除、打标签（专业模式）。

### 发布看板（Ctrl+3）

- 指标卡片：总任务、待发布、失败、TG、健康问题。
- 点击卡片可跳转并自动筛选相关任务。
- 简易模式下不显示「智能下一步」卡片（与底部引导栏去重）。

### 发布向导（Ctrl+4）

- 统一进入 **任务工作台内嵌向导**，不再打开独立向导页。

### 运维中心（Ctrl+5，专业模式）

- 活动日志、陈旧任务、重复扫描、批量导出统计等。
- 简易模式下默认隐藏，可通过命令面板或切换专业模式进入。

### 设置

- 外观：亮色/暗色主题、中/英文。
- 工作区：工作室名称、路径习惯。
- 集成：TG Bot、发布模板、夜间脚本等。
- 可导出/导入机器配置与快捷键映射。

## 6. 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 命令面板（简易模式为 8 项常用操作） |
| `Ctrl+Shift+P` | 切换简易 / 专业模式 |
| `Ctrl+1` ~ `Ctrl+5` | 快速处理 / 任务 / 看板 / 向导 / 运维 |
| `F5` / `Ctrl+R` | 刷新当前页 |
| `Ctrl+Enter` | 运行完整流水线（任务页） |
| `Ctrl+S` | 保存发布链接（任务页） |
| `Esc` | 关闭提示条 / 跳过引导 |

更多快捷键见程序内 **帮助** 按钮，或 `CHANGELOG.md`。

## 7. 命令行版（可选）

路径：`PLCPak.Cli\`

```bat
PLCPak-CLI.bat -Preview -Path "D:\游戏目录" -NoGui
PLCPak-CLI.bat -JobList -Json
```

`预览样本.bat` 可用自带广告样本快速测试。完整参数见 `winui/README.txt`（开发文档）或运行 `PLCPak-CLI.bat -Help`。

## 8. 数据备份与升级

**建议备份：**

- `workspace/`（任务与产出）
- `data/studio-config.json`（工作室配置）
- `workspace/ui-prefs.json`（界面偏好）

**升级步骤：**

1. 备份上述目录。
2. 解压新版 ZIP 到新文件夹，或覆盖旧版（保留 `workspace`）。
3. 用 `启动 PLCPak.bat` 启动验证。

## 9. 常见问题

**Q：启动闪退或报错？**  
A：查看 `logs/plcpak-startup.log` 或 `PLCPak.WinUI/logs/plcpak-startup.log`，将日志反馈给维护者。

**Q：暗色主题下界面发白？**  
A：请使用 v1.0.74 修复版；旧包请重新下载。

**Q：压缩很慢？**  
A：检查 `data/7-Zip-Zstandard/7z.exe` 是否存在；大文件建议在设置中调整线程与字典。

**Q：TG 发送失败？**  
A：在设置中检查 Bot Token、Chat ID；任务内需已生成文案并填好链接。

**Q：如何更新广告样本清单？**  
A：运行 `scripts/更新广告清单.bat`（需保持 `data/AD-Samples` 结构完整）。
