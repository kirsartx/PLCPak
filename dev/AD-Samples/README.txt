【广告样本文件夹使用说明】

本文件夹用于存放广告文件样本。
打包程序将根据此文件夹中的样本，自动检测并清除待压缩文件夹中的广告文件。

═════════════════════════════════════════════════════════════

【使用方法】

1. 将广告文件放入此文件夹（AD-Samples）
2. 支持任意类型的广告文件（.txt, .url, .bat 等）
3. 拖入文件夹到打包工具时，会自动检测和清除匹配的广告文件

═════════════════════════════════════════════════════════════

【检测规则】

程序会使用以下方式识别广告文件：

✓ 文件名完全匹配（不区分大小写）
✓ 文件 SHA256 哈希值匹配（用于内容相同但文件名不同的情况）

═════════════════════════════════════════════════════════════

【示例】

如果样本文件夹包含：
  - 广告.txt
  - 推广链接.url
  
当压缩某个文件夹时，所有包含这些文件的路径都会被检测并清除。

═════════════════════════════════════════════════════════════

【日志输出】

清除广告时，会在日志中显示：
  ✓ 检测到的广告文件数量
  ✓ 已清除的广告文件列表
  ✓ 清除位置（完整路径）

═════════════════════════════════════════════════════════════

【注意事项】

⚠ 请确保样本文件确实是广告文件，避免误删重要内容
⚠ 文件名匹配不区分大小写
⚠ 哈希匹配可以识别重命名的广告文件
⚠ 样本文件夹为空时，不会执行广告检测
